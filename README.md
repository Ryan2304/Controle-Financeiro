# IR Planner (Excel)

Planilha em Excel para **organizar e controlar as informações necessárias para a Declaração de Imposto de Renda** (PF), com foco em praticidade, padronização e navegação rápida.  
Ideal para uso pessoal e também como **projeto de portfólio**.

---

##  Visão geral

O **IR Planner** centraliza os principais dados usados na declaração, reduzindo retrabalho e evitando erros comuns de preenchimento.  
A ferramenta foi construída no Excel com **validações**, **formatação automática**, **cálculos**, **hiperlinks** e **navegação por botões**.

---

##  Funcionalidades

###  Interface e navegação
- Menu lateral com acesso rápido às seções:
  - **Titular**
  - **Informes**
  - **Notas**
- Botões de navegação (inclui **Próximo** e **Anterior**).
- Design moderno (tema escuro + verde premium), pensado para uso diário.

###  Proteção e segurança de edição
- Planilha **protegida** para evitar alterações em áreas que não precisam ser editadas.
- Usuário preenche apenas os campos necessários (demais áreas ficam bloqueadas).

###  Validações e padronização
- Campos com **validação de dados** para reduzir erros de digitação.
- Padronização automática de formatos:
  - **CPF**
  - **CEP**
  - **Datas**
  - **Valores (R$)**
- Lista de **bancos válidos** com validação (seleção guiada).

###  Cálculos e automações
- Totais automáticos (ex.: soma de valores por bancos/entradas).
- Organização para anexos/controle de comprovantes.

---

##  Estrutura da planilha

### 1) Dados do Titular
Campos para cadastro de pessoa física e informações essenciais:
- Nome, CPF, nascimento, título de eleitor, cônjuge
- Endereço (rua, CEP), telefone, celular, e-mail
- Indicadores (ex.: alterações, dependente cônjuge, residente no exterior)

### 2) Informes de Rendimentos Bancários
Área para lançar os valores por banco:
- Banco (com validação)
- Valor atual
- Campo/controle de anexo
- Total consolidado

### 3) Notas bancárias / Extrato / Holerites
Tabela para entradas mês a mês:
- Data
- Categoria
- Valor
- Estrutura preparada para uso contínuo e organização anual

---

##  Como usar

1. Baixe o arquivo `.xlsx` deste repositório.
2. Abra no Excel (recomendado: Microsoft Excel Desktop).
3. Use o menu lateral para navegar:
   - **TITULAR** → preencha seus dados
   - **INFORMES** → cadastre bancos e valores
   - **NOTAS** → registre entradas/holerites/extratos ao longo do ano
4. Anexe/organize seus comprovantes conforme a estrutura da planilha.

> Dica: preencha somente os campos em verde (áreas editáveis).
